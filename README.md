SaveMyNotes
===========

This application was developed to meet the needs below. But it is simple enough to serve as a reference for the future (copy + paste).
Features:
>	* Custom list adapter
>	* Custom list empty view
>	* Simple Database 
>	* CRUD using DAO
>	* Menus.

Save My Notes!

Objectivos do teste:

    Utilização das ferramentas de gestão de dominio (DAL + ORM) para persistência de informação

Peço-vos que façam uma aplicação que recebe uma nota e a grava localmente na base de dados SQLite do Android. Sempre que a aplicação é carregada esta deve fazer load das notas que existem no sistema e permitir a criação de novas notas, apagar notas e actualizar notas.

As operações de apagar (delete) e actualizar (update) surgem ao utilizador como 2 opções no menu de contexto que aparece com um “long click” em cima da linha onde a nota aparece na List View que está no layout.

Para a criação desta aplicação irão usar o layout mostrado em baixo, que pode ser descarregado aqui:
